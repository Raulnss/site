III.16 – PROJETO DE APLICAÇÕES PARA WEB

Função: Estudo, Planejamento, Desenvolvimento e Gerenciamento de Projetos
COMPETÊNCIAS HABILIDADES BASES TECNOLÓGICAS
1. Modelar aplicações web respeitando os padrões sugeridos pela World Wide Web Consortium (W3C) abordando seus aspectos principais como interação com usuário.
2. Desenvolver, aprimorar e publicar um sistema web de acordo com a análise de requisitos.
1.1. Conhecer os padrões para a criação e a interpretação dos conteúdos para a web, desenvolvido pela W3C.
1.2. Identificar as necessidades dos usuários e propor o desenvolvimento de um website.
1.3. Identificar técnicas eficientes de comunicação do desenvolvimento de websites.
2.1. Utilizar técnicas de modelagem de dados.
2.2. Aplicar as técnicas de modularização, especificação e verificação de websites.
2.3. Utilizar ferramentas de apoio ao desenvolvimento de websites.
2.4. Entender a orientação a objetos na construção de websites.
